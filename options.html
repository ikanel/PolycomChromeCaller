Available lines:

Specify the number of lines available in your Polycom phone

Phone Address

Specify the phone address including protocol and without trailing slash. Example: http://192.168.0.100

Phone push User

Specify the username for the Polycom push service. To define the username login into the Polycom Web Configuration Utility. Just type the phone IP address in your browser. The default admin password for the utility is 456. Go to the settings/applications/push, enable all push messages, specify username and password.

Phone push Password

Specify the push service password

Phone number rewriting

It is possible to prepend the phone number with the country code if it is was not defined. If you want to use this feature - specify the phone line to prepend, country code and check an appropriate checkbox.
